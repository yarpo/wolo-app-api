# Wolo-app-api

REST API for web Application for Regional Volunteer Centre in Gda≈Ñsk, which helps volunteers find and partake in events.

## Tech Stack

- Gradle
- SpringBoot
- Postgresql
- Liquibase
- REST Assured

## Run Locally

Clone the project

```bash
  git clone https://github.com/yarpo/wolo-app-api.git
```

Go to the project directory

```bash
  cd wolo-app-api
```

Run

```bash
  ./gradlew bootRun
```

Mapping

- http://localhost:8080/persons
- http://localhost:8080/health



## Running Tests

To run tests, run the following command

```bash
  ./gradlew test
```
